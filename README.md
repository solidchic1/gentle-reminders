# gentle-reminders

bitch, drink some water!


A lightweight, no-account browser-based reminder tool for water, meals, sleep, and wake-up schedules.

## what it does

Open the page, set your schedule once, hit save. As long as the tab stays open, it fires browser notifications when it's time to drink water, eat, wind down, or wake up. Settings persist in localStorage so they reload automatically on your next visit.

No backend. No accounts. No data leaves your browser.

## how to use

1. Open the page in any modern browser
2. Allow notifications when prompted
3. Fill in your schedule
4. Hit **save my schedule**
5. Keep the tab open (pin it for best results)

## deploy your own

Static HTML — no build step needed.

1. Fork or clone this repo
2. Connect to [Netlify](https://netlify.com) via GitHub
3. Deploy as-is — no build settings required
